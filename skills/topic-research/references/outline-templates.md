## 模板类型

### 1. authoritative — 权威揭秘型

**适用场景**: 专业知识科普、辟谣纠正误区、行业内幕揭露

**内容结构**:

1. 权威背书建立 — 引用专家/数据建立可信度
2. 常见误区揭示 — 点出读者常见认知错误
3. 专业原理解析 — 用通俗语言解释底层逻辑
4. 实用方法指导 — 提供可操作的正确做法
5. 价值总结升华 — 提炼核心收获，号召分享

**典型标题公式**:

- `专家揭秘：[话题]的N个关键秘密`
- `你以为的[话题]，其实90%的人都错了`
- `[话题]真相：权威数据告诉你答案`

**开头钩子风格**:

- 惊人统计数据开场（"90%的人都…"）
- 权威机构/专家引用
- 反常识陈述引发好奇

**互动设计**:

- 在误区部分设置"你中招了吗"投票
- 结尾追问"你之前是这样认为的吗？"

### 2. comparison — 对比评测型

**适用场景**: 产品/方案选择指南、选择困难解决方案、横向测评

**内容结构**:

1. 对比对象介绍 — 简要介绍各选项背景
2. 多维度对比分析 — 价格/效果/适用人群等对比表格
3. 优劣势客观评价 — 不偏颇地分析各自优缺点
4. 适用场景建议 — 根据不同需求给出推荐
5. 选择指南总结 — 一句话选择原则

**典型标题公式**:

- `[A]vs[B]：一张图看懂核心区别`
- `[话题]大对比：选哪个才不后悔？`
- `深度测评：[N种选项]哪个最值？`

**开头钩子风格**:

- 选择困难症共鸣（"每次选择都纠结…"）
- 直接给出结论再展开说明
- 用户真实评价引用

**互动设计**:

- 在对比表格后设置"你会选哪个"投票
- 邀请读者分享自己的使用经验

### 3. cultural — 文化故事型

**适用场景**: 传统文化传播、品牌故事挖掘、历史人文解读

**内容结构**:

1. 历史背景铺垫 — 用生动细节还原历史场景
2. 文化故事展开 — 讲述核心文化故事/人物/事件
3. 精神内核挖掘 — 提炼文化背后的精神价值
4. 现代价值连接 — 与当代生活/价值观建立连接
5. 情感共鸣升华 — 引发民族自豪感或情感认同

**典型标题公式**:

- `[话题]背后的文化密码：千年传承的现代表达`
- `那些被遗忘的[话题]故事`
- `一[话题]，读懂中国人的[精神/智慧/情感]`

**开头钩子风格**:

- 沉浸式历史场景描写
- 不为人知的历史细节揭露
- 当代现象反差引入历史

**互动设计**:

- 引导读者分享自己与[话题]的故事
- 文化知识问答互动

---

### 4. practical — 实用干货型

**适用场景**: 技能教程、解决方案指南、高效方法分享

**内容结构**:

1. 问题痛点提出 — 引发读者共鸣的具体场景
2. 原因原理解析 — 简短说明为什么会出现这个问题
3. 具体方法步骤 — 可操作的步骤（3-7步，每步配图）
4. 注意事项提醒 — 常见错误和避坑指南
5. 效果预期说明 — 按照方法做能达到什么效果

**典型标题公式**:

- `3分钟学会：[话题]的实用指南`
- `[话题]终极攻略：N步解决所有问题`
- `收藏！[话题]最全教程（附步骤图）`

**开头钩子风格**:

- 痛点场景描写（"你是不是也遇到过…"）
- 强调时间价值（"3分钟看完，节省N小时"）
- 承诺具体收益

**互动设计**:

- 步骤清单格式方便截图保存
- 结尾邀请分享使用效果
- "学到了吗？点赞告诉我"

## 使用说明

调用 topic-research Skill 的大纲步骤时，通过 template 参数指定模板类型：

- authoritative（权威揭秘型）
- comparison（对比评测型）
- cultural（文化故事型）
- practical（实用干货型）

Claude 代理在生成大纲时，应参考对应模板的结构、标题公式和互动设计要点，
生成符合该模板逻辑的完整内容框架 JSON。
