# 标题 CTR 心理学

> 配合 [SKILL.md](../SKILL.md) 使用。环节 3（标题 CTR 层）的方法论详解。

## 为什么标题是第一战场

订阅号打开率近年持续走低（行业普遍跌破低个位数百分比，私域衰退），文章要活下来越来越依赖"看一看/推荐"公域入口。而无论私域还是公域，**标题是用户和内容的唯一中介**——标题失败，内容再好没人点开。标题的核心任务：在 0.5 秒内让读者产生"这跟我有关/我得看看"的冲动。

## 6 种好奇心缺口（标题的点击引擎）

好奇心缺口 = 标题暗示"有个你不知道/反直觉的答案"，读者必须点开才能补全。6 种制造缺口的手法：

| 手法 | 机制 | 示例骨架 |
|------|------|----------|
| **信息缺口** | 暗露一半，藏一半 | "……的真相，大多数人不知道" |
| **反常识** | 打破默认认知 | "你以为 X？其实 Y" |
| **悬念结果** | 给结果不给过程 | "我是如何用 X 天做到 Y 的" |
| **对比反差** | 两个对立项并列 | "A 和 B 的差距，不在…" |
| **身份点名** | "说的就是你" | "30+ 的人都该看看…" |
| **数字杠杆** | 具体数字制造可信与好奇 | "90% 的人都在错的…" |

合规提醒：好奇心缺口靠 **认知悬念**，不靠夸大承诺。禁止用极限词制造虚假缺口（"史上最强/100% 有效"既违规又伤完读率——读者点进来落差大，直接划走）。

## 标题公式库

| 公式 | 模板 | 何时用 |
|------|------|--------|
| **数字型** | N 个…/N 天…/N% 的人… | 清单、教程、数据型 |
| **痛点型** | 别再…了/为什么你越…越… | 解决方案、避坑 |
| **反差型** | A 不是 B，而是 C | 反常识、揭秘 |
| **悬念型** | ……的真相/原来…… | 揭秘、科普 |
| **共鸣型** | ……的人都懂/致…… | 情感、身份认同 |
| **利益型** | 掌握…，告别…/让你… | 实用价值、教程 |

一个标题常组合 2 个公式（如 数字 + 痛点）。

## 标题合规边界（硬性）

- **禁极限词**：最/第一/顶级/完美/100%/根治/特效/国家级/首选（无认证）/终极/史无前例。完整词库见 `content-writing` skill 的 `references/prohibited-words.md`。
- **禁虚假承诺**：零风险/包过/绝对/一定赚钱。
- **禁标题党误导**：标题承诺的内容正文必须有，否则完读率崩塌 → 推荐归零（合规与算法同向）。
- **核心词前置**：核心关键词放标题前半部分（前 10 字），利于搜一搜与快速扫读。
- **字数**：22–25 字为黄金区间（超 64 字会折叠）。核心利益点必须落在折叠线之前。

## 3 变体打分表（环节 3 选优用）

生成 3 个不同公式的标题变体后，按下表打分（每维 0–2 分，满分 12），**选最高分**为最终标题：

| 维度 | 0 分 | 1 分 | 2 分 |
|------|------|------|------|
| 好奇心缺口 | 纯描述无缺口 | 有轻微悬念 | 强缺口（必须点开才知） |
| 情绪强度 | 无情绪 | 有情绪但弱 | 强情绪（焦虑/好奇/共鸣） |
| 数字杠杆 | 无数字 | 有数字但非核心 | 数字是核心抓手 |
| 合规性 | 含极限词（0，弃用） | 边缘但合规 | 干净合规 |
| 核心词前置 | 核心词在后半/无 | 中部 | 前 10 字 |
| 字数 | <15 或 >30 | 15–21 或 26–30 | 22–25 |

打分要点：

- **合规性为一票否决**：含极限词的变体直接 0 分弃用，不得因为其他维度高而保留。
- 其余维度看是否服务于"0.5 秒抓点击"这一个目标。

把 3 个变体 + 打分记录写进 `seo-result.md`，便于审计复盘。

## 标题-封面-摘要三位一体协同

三者必须 **互相强化同一个钩子**，不能各说各话：

- **标题**给利益点/悬念（文字钩子）
- **封面**给主视觉冲击（视觉钩子，是否带短文字由 `article-cover-design` 的受控文字策略决定）
- **digest（摘要）**前半句再放大同一利益点/情绪（放大钩子）

读者在"看一看"刷到：先被封面视觉钩住 → 读到标题文字钩子 → 摘要补刀 → 点开。三者指向同一个"这跟我有关/我得看看"，合力最大。

**协同检查**：最终标题的核心利益点 == 封面主视觉隐喻 == digest 前半句利益点。三者偏离则 CTR 打折——这是环节 3 的硬性自检项。

### digest 按 CTR 优化的写法

`seo-optimization` 的 digest 偏"含关键词 + 概括"。本环节在其上叠加"前半句制造点击冲动"：

- 前 30 字抛利益/悬念/反差（"90% 的人都在错的…/这篇帮你避开…"）。
- 后半句再落到核心价值与关键词，兼顾搜一搜。
- 长度仍守 50–120 字。
