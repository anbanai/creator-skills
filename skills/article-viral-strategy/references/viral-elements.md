## 底层共识（2025+）

微信公众号正从"订阅优先"转向 **算法推荐 + 社交裂变** 双引擎：

- **算法推荐**：完读率是进推荐池的核心门票。标题党 → 完读率低 → 零推荐。
- **社交裂变**：转发是流量的第二引擎，而转发由 **社交货币** 驱动——读者转发一篇文章，本质是在"借这篇文章表达他自己"。

四要素对应关系：**标题 → 打开率，结构 → 完读率，内容 → 分享率，视觉 → 停留时长**。再加互动诱因 → 评论率。五个全打满，才有爆款。

## 维度 1：选题社交货币（驱动转发率）

### 为什么是社交货币

读者为什么会转发？不是因为文章"好"，而是因为转发这篇文章能让他在朋友圈/好友眼里 **显得更某样**（更懂行、更有品味、更关心家人、更会生活）。文章提供的这种"可借来自我表达"的价值，就是社交货币。社交货币属性越强，转发率越高。

### 五种社交货币类型

判定本选题给读者的是哪种货币（可叠加），决定写法与转发钩子：

| 货币类型 | 读者转发的潜台词 | 适合选题 | 转发钩子写法 |
|----------|------------------|----------|--------------|
| **谈资** | "这个有意思/新鲜，想看看别人怎么看" | 行业揭秘、反常识、热点解读 | 强调"内幕/真相/第一次听说" |
| **实用工具** | "这个有用，要收藏/转给需要的人" | 教程、清单、避坑、对比 | "建议收藏/转给 XX" |
| **身份认同** | "这就是我/我们这群人" | 群体画像、行业自嘲、代际共鸣 | 点名人群（"打工人/30+女生/…"） |
| **自我表达** | "说得就是我心里想的，替我说出来了" | 情感共鸣、观点输出、价值观 | 替读者说出他平时说不出口的话 |
| **利他分享** | "这个对我关心的人有用，转给他" | 健康、育儿、理财、安全 | 明确受益对象（"转给父母/伴侣"） |

### 选题判定要点（环节 1 用）

选定选题后，回答三问，写进「爆款策略」段：

1. **货币类型**：本文主要给读者哪种货币？（≥1 种，可叠加）
2. **核心情绪**：本文击中的主情绪是什么？（焦虑/好奇/共鸣/愤怒/感动/认同/释然/紧迫…）
3. **时效借势**：是否搭季节（开学/年终/换季）、节日、热点事件窗口？最佳发布时机？

判定标准：货币类型清晰 + 情绪明确 + 受众能"对号入座" → 转发潜力高；反之主题宽泛、只靠产品本身、情绪模糊 → 转发潜力低。

### 时效借势（无实时热搜 API 时的判断法）

本管线无实时热搜/微信指数 API，时效借势靠**结构性判断**而非实时数据：

- **季节性**：选题是否天然绑某季节/节点（换季养生、年终复盘、开学季）？提前 3–5 天布局。
- **长热点**：选题是否搭一个持续数周的社会议题/政策/现象？借势但要给独家角度，避免和海量同质文撞车。
- **常青**：若选题非常青也非热点，就主打"情绪/实用"货币，不强行蹭时效。

借势判断写进「爆款策略」段的「时效借势」字段；无时效性写"常青"。

## 维度 2：分享率内容（驱动转发率 / 收藏率）

读者花时间看完，必须觉得"值得"——值得才会转发、值得才会收藏。

### 三种核心价值（择一为主）

每篇文章必须有 **一个** 清晰的核心价值类型，不要样样都沾样样松：

| 价值类型 | 读者感受 | 内容特征 |
|----------|----------|----------|
| **情绪价值** | "被说中了/被治愈了/热血了" | 共鸣、抚慰、激励、宣泄 |
| **信息价值** | "原来如此/我不知道的事" | 新知、揭秘、深度解读、独家视角 |
| **实用价值** | "这个我能立刻用上" | 方法、步骤、清单、工具、避坑 |

主价值类型在「爆款策略」段标注，贯穿全篇写法（情绪类重共鸣与金句；信息类重新知与反差；实用类重可操作与收藏诱因）。

### 收藏诱因（驱动收藏率）

收藏 = 读者判断"以后还要回看"。天然收藏诱因：

- **清单/列表**（"7 个习惯/5 个工具"）
- **步骤/流程**（"第一步…第二步…"）
- **对比表**（A vs B 怎么选）
- **避坑指南**（"千万别踩这 3 个坑"）
- **速查/模板**（可直接套用）

实用价值类天然带收藏诱因；情绪/信息类也要想办法"埋"一个可保存的信息点（如结尾一张要点卡、一个可复用的金句），拉高收藏率。

### 价值密度硬要求

- 每段至少一个具体信息点（数字/场景/方法/案例/人物/冲突），不写纯感受堆砌。
- 删掉读者"看完就忘"的水段——水段会拉低完读率。

## 爆款公式速查（结构骨架）

四种经过大量验证的爆款结构，按选题类型选一种做骨架：

| 公式 | 结构 | 适用 |
|------|------|------|
| **痛点 + 解决方案** | 痛点描述 → 具体方法 → 即时效果 | 教程、攻略、方法论 |
| **反差 + 揭秘** | 打破认知 → 真相分析 → 实操建议 | 知识科普、行业揭秘、反常识 |
| **合集 + 测评** | 选品逻辑 → 对比维度 → 推荐结论 | 好物、产品对比、清单 |
| **经历 + 复盘** | 真实经历 → 踩坑总结 → 经验提炼 | 个人成长、职业分享、情感 |

## 爆款 vs 普通对比

| 维度 | 爆款 | 普通 |
|------|------|------|
| 标题 | 情绪词/数字/悬念 + 核心词前置 | 纯信息描述 |
| 开头 | 场景/故事/痛点/金句代入 | "今天来分享…" |
| 内容 | 每段一个可操作/可记忆点 | 泛泛而谈 |
| 结尾 | 具体互动问题 + 收藏钩子 | "欢迎关注" |
| 价值感 | "这篇我得收藏/转给 XX" | "看完就忘了" |
