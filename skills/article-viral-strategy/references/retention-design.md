# 完读率与互动设计

> 配合 [SKILL.md](../SKILL.md) 使用。环节 2（写作爆款硬性要求）的方法论详解。完读率是微信推荐池的**核心门票**，读者从点开到读完，每一段都在"要不要划走"。本文件给五件武器：开头、节奏、情绪、金句、诱因。

## 1. 黄金三秒开头（6 种钩子）

前 100 字（手机首屏）决定读者留不留。**禁用零钩子开场**（"今天来分享…"/"随着…的发展…"/"大家好…"/"在我们的生活中…"）。

6 种验证有效的开头钩子：

| 钩子 | 怎么写 | 为什么有效 |
|------|--------|------------|
| **金句定调** | 开篇一句短而锐的观点/判断，定全篇基调 | 调动情绪、引发共鸣或争议 |
| **痛点共鸣** | 直接说出读者正经历的痛 | "说的就是我" → 留下来看解法 |
| **热点切入** | 借当下热点/事件引出主题 | 借势已有注意力 |
| **反问悬念** | 抛一个读者答不上来的问题 | 制造信息缺口 |
| **故事场景** | 一个具体的人/场景/瞬间 | 画面感代入，降低阅读门槛 |
| **反常识** | 开篇就打破一个默认认知 | "不可能吧？" → 读下去验证 |

**选钩子原则**：与核心情绪一致。

- 情绪共鸣类 → 金句 / 痛点
- 知识科普/揭秘类 → 反常识 / 反问
- 实用教程类 → 痛点 / 故事场景

## 2. 情绪弧曲线

平铺的情绪留不住人。全篇情绪要有 **起伏**，至少：

- **一个共鸣峰**：让读者"对，就是这样"的高点（通常在前 1/3，承接开头钩子）。
- **一个转折/释然**：给情绪一个出口或升华（通常在后 1/3，引向结尾）。

避免两种极端：全程同一种情绪强度（要么全程平、要么全程煽），读者都会疲劳。情绪弧的起伏本身就是留人机制——读者想知道"后来怎样了/怎么破的"。

## 3. 金句密度

金句 = 能被单独摘出来发朋友圈的短句（短、锐、有观点/画面）。作用：读者读到会"想截图/想转发这句" → 直接拉高转发率。

- **密度**：每千字 ≥3 句金句。
- **特征**：短（≤20 字）、有判断/比喻/反差、口语化不端着。
- **位置**：开头定调一句、各章节核心论点处、结尾升华一句。

> **与 humanizer 的关系**：humanizer 去 AI 味时会把"空洞拔高"当 AI 痕迹清除（如"在这个瞬息万变的时代…"）。金句必须是 **有具体信息含量的锐句**（带判断/比喻/数字/画面），不是空拔高。二者不冲突——人味金句保留，AI 空话清除。写金句时自检：删掉这句，信息量有没有实质损失？有 = 好金句；没有 = 空拔高，重写。

## 4. 移动端完读率节奏

- **段落短**：单段 ≤4 行（手机屏），长段落必须拆分。
- **小标题有信息量**：每个 `##` / `###` 是一个"可扫读的利益点"，不是"关于 X"这种空标题。
- **留悬念**：关键转折处不要一次说尽，引导往下读。
- **句式交替**：长短句交替，避免连续相同句式开头（`content-writing` 已有此要求，爆款层强化）。

## 5. 互动诱因设计（转发/收藏/评论各 ≥1，合规）

互动是算法判活跃的核心信号。但 **合规红线**：不靠抽奖、福利、关注换资料、回复关键词、扫码、加微信、进群、跳小程序/其他公众号/视频号/外链交易诱导（平台打压、伤账号）。要用"内容自然激发"的诱因。

### 转发诱因（驱动转发率）

- 给读者"转发 = 表达我自己"的素材：一句替他说出心里话的金句、一个身份认同的标签。
- 明确受益对象时点出（"转给那个总说没时间的朋友"）。
- 货币类型决定钩子（见 viral-elements.md 社交货币表）。

### 收藏诱因（驱动收藏率）

- 埋一个可保存的信息点：清单/步骤/对比/避坑/速查（见 viral-elements.md 收藏诱因）。
- 结尾可放一张"要点总结卡"提示收藏。

### 评论诱因（驱动评论率）

- **低门槛互动**：选择题/填空 > 开放题 > 无引导。
- **争议留白**：留一个有讨论空间的观点让读者站队。
- **经验交换**：邀请读者分享自己的经历/做法。
- 结尾给 **一个具体的、好回答的问题**（不是"欢迎留言"）。

| 好（合规有效） | 差/违规（伤账号） |
|----------------|-------------------|
| "你踩过哪个坑？评论区聊聊" | "留言抽奖" |
| "A 和 B 你站哪个？" | "关注领资料" |
| "你的经历是怎样的？" | "转发截图领红包" |
| "这份清单就在文内，先收藏慢慢看" | "回复关键词领取完整版" |

诱因必须与正文价值 **自然连接**，硬塞的互动反而尴尬、伤完读。
