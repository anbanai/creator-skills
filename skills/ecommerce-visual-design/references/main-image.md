# 主图套设计规范（CTR 之战）

主图是电商曝光→点击的第一战场，**点击率（CTR）优先于一切**。默认 5 张一套（京东 6 张、微信小店 3-9 张），比例与尺寸按 `target_platform`（见 `ecommerce-platform-specs/references/platforms.md`）：淘宝天猫/京东/微信 `1:1:2K`（生产 800×800），抖音/小红书服饰类 `3:4:2K`（竖版）。移动端首屏可读。

> 设计方法论（字体/色彩/构图/信息层级/移动端）见 [design-principles.md](design-principles.md)——主图的每张都要用上「主角-配角-钩子」三级层级与三分法构图。

## 5 张结构与 CTR 模板（默认；按品类/平台微调）

### ① 点击主图（CTR 之王）
- **目标**：3 秒法则——买家扫过即懂「这是什么、解决什么、凭什么点我」。
- **信息层级**（用 design-principles 的 5 变量实现）：产品主体占图 50-65%（最大、黄金分割点）→ 第一卖点文案（图宽 ~14% 字号、思源黑体加粗）→ 价格/促销钩子（红底白字角标，小而醒目）。
- **必含**：第一卖点（来自 copywriting 排序第 1）+ 真实场景或纯白高品质产品图 + 一个钩子（限时/赠品/包邮，须合规）。
- **禁**：信息过载（>1 主文案+2 辅文案）、极限词、与产品不符的场景、牛皮癣（文字压主体/铺满背景）。
- **首屏安全区**：核心卖点与主体落中心 80%，**避开右下价格标位**（淘宝/京东/抖音价格标常在右下）。

### ② 细节/材质特写
- 微距质感、工艺细节、材质卖点；建立「品质值得这个价」的信任。
- **点名保真**：与产品图清单中细节/材质那张（如「第 3 张(细节)」）完全一致。

### ③ 使用场景
- 代入感：谁/何时/何处用；场景化利益，扩大买家自我认同。
- 抖音/小红书尤其重场景情绪代入。

### ④ 对比/第二卖点
- 与痛点/旧方案对比（前后/有无，有据不贬损具体竞品），或第二核心卖点的可视化。
- 对比用强对比色/明暗差制造冲击（design-principles 互补色点缀）。

### ⑤ 品牌/资质/服务承诺
- 品牌背书、检测认证图标、售后（7 天无理由/运费险/质保）——闭环信任，降低犹豫。
- 京东把这格做得更突出（参数/服务标识导向）；3C/家电/食品/美妆的资质图标须有据。

## 平台差异

| 平台 | 套系要点 |
|---|---|
| **淘宝天猫** | ① 强制/建议白底（天猫更严）；牛皮癣禁忌；移动优先；钩子合规 |
| **京东** | **6 张**含参数/资质格；白底强制；服务标识突出；参数详尽 |
| **抖音** | 1:1 + 3:4 双套；**首图必须实物图**；信息流前 0.5 秒强冲击；禁全屏水印；主图不得完全相同 |
| **小红书** | 3:4 竖版；弱硬广/素人感；封面标题钩子强 |
| **微信小店** | 1:1；3-9 张不得重复；信任背书调性 |

## 视觉工艺要点

- **卖点可视化**：抽象卖点→可视符号（长续航→电量条/时钟；轻→羽毛/比例；防水→水珠/IPX 等级）。把卖点符号写进 prompt（见 design-principles「翻译成 prompt」）。
- **强对比**：明暗/冷暖/前后对比制造视觉冲击。
- **留白**：主体四周留白，避免压迫；文字不压在产品关键部位——留白越足越高级。
- **一致性**：5 张同色系/版式/字体（同 `$STYLE`）；产品锚点参考用于①，细节/场景图保持同一商品（点名保真）。
- **白底诚实声明**：白底图的纯白 RGB(255,255,255) 与 <300KB 是投放后期约束，生成做到「白底为主、主体居中、无牛皮癣」，纯白与压体积由后期完成。
