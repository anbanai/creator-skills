# 封面 / 类目 Banner 设计规范（品牌氛围与入口）

封面/类目 banner 承担店铺/活动/类目入口的视觉定调，`16:9:2K`（PC/类目顶图）或 `3:4:2K`（移动端竖版入口）。**氛围优先于信息密度**。

> 设计方法论见 [design-principles.md](design-principles.md)：banner 用 Z 型视线引导（logo/钩子左上→产品中→CTA 右下）、宽幅留白、品牌色统一。

## 类型与目标

- **店铺/品牌 banner**：品牌氛围 + 主推品 + slogan，定调品牌质感。
- **类目 banner**：类目名称 + 代表品 + 导购暗示，引导进入类目。
- **活动 banner**：活动主题 + 利益点（满减/赠品）+ 时间，**合规真实**（不虚构原价/倒计时/限量）。

## 视觉工艺要点

- **氛围优先**：宽幅构图，留白与场景感强于主图；弱化密集信息。
- **品牌一致性**：延续 `$STYLE` 与产品锚点；logo/品牌色规范统一（点名保真 logo 文字与颜色）。
- **文字精简**：1 主标题（≤10 字，思源黑体/宋体按调性）+ 1 副标题/利益点；字号大、留白足；信息层级用 design-principles 的主角-配角实现。
- **主体位置**：产品主体偏左/右三分之一（三分法），文字避让；符合构图原理。
- **移动/PC 双适配**：16:9 适合 PC/类目顶图，3:4 适合移动端竖版入口——按投放位选尺寸。
- **平台差异**：淘宝/京东类目顶图偏 16:9 规范感；抖音/小红书偏 3:4 场景氛围；微信私域偏信任背书。
- **禁**：极限词、与活动不符的虚假利益、遮挡主体的文字、牛皮癣。
