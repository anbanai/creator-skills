# 分享图设计规范（站外引流）

分享图用于站外社交/私域引流，`1:1:2K` 或 `3:4:2K`（按目标平台/渠道）。目标：引发好奇→点进商品。

> 设计方法论见 [design-principles.md](design-principles.md)：弱广告感=克制信息+留白+真实质感，靠构图与色彩而非堆卖点。

## 视觉工艺要点

- **社交钩子**：疑问/对比/利益前置，引发好奇（如「这件事居然……」「对比后我选了它」）。
- **弱广告感**：比主图更接近 UGC/种草感但仍保持商业品质；避免一眼硬广——少 logo、少促销角标、多留白与真实场景。
- **主体突出**：产品或使用结果是视觉中心；文字 1 钩子句 + 1 利益点，不超 2 行。
- **一致性**：产品与主图/详情同一件（点名保真）；可复用锚点参考或共享 `$STYLE`。

## 分渠道口吻（按分享渠道调整文案与调性）

| 渠道 | 口吻 | 钩子风格 |
|---|---|---|
| **小红书** | 生活方式 + 真实感 + 素人感 | 疑问式/体验式标题（「被我种草了…」「终于找到…」），3:4 竖版 |
| **微信（朋友圈/社群）** | 信任背书 + 私域亲密感 | 熟人推荐口吻（「自用回购 N 次」），1:1 |
| **抖音** | 强冲击 + 节奏 + 情绪 | 利益/反差前置短句（「别再买错了」），3:4/9:16 |
| **淘宝/通用** | 利益 + 卖点 | 对比/优惠钩子（「同款它更划算」），1:1 |

- **合规**：不夸大、无极限词；引导语符合平台规则——**微信禁诱导分享/裂变**（「转发得奖」「帮砍」类违规），小红书软广须标、禁明显导流外部。
