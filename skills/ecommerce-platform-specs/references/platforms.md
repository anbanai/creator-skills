## 标识符契约（value ↔ 中文 ↔ 章节）

agent 收到的 `target_platform` 是 `value`（来自建任务时 UI 选项 / 任务配置）。**先按下表归一化到章节**，未知值 → 走「通用」。

| `target_platform` 值 | 中文 | 对应章节 | 主图默认比 | 关键差异 |
|---|---|---|---|---|
| `taobao` | 淘宝 / 天猫 | 淘宝天猫 | 1:1 | 5 张主图；天猫白底/品牌 LOGO 更严，章内分述 |
| `jd` | 京东 | 京东 | 1:1 | **6 张**主图；白底强制类目广；参数/服务标识导向 |
| `douyin` | 抖音电商 | 抖音 | **1:1 或 3:4 双套** | 首图必须实物图；信息流强冲击；竖屏优先 |
| `xhs` | 小红书电商 | 小红书 | **3:4** | 原生竖版；生活方式/弱硬广；素人感 |
| `wechat_store` | 微信小店 | 微信小店 | 1:1 | **新增**；3-9 张主图；微信生态合规极严 |
| `general` / 未知 / 空 | 通用（多平台兼容） | 通用 | 1:1 | 兜底，便于跨平台裁切复用 |

> ⚠️ **契约现状**：`taobao`/`jd`/`douyin`/`xhs` 当前可选；`wechat_store` 暂未进 Studio UI enum（agent 收到该值或中文「微信小店」时仍按「微信小店」章节处理，UI 落地后即端到端可用）。天猫暂与淘宝同值 `taobao`，章内给出天猫增量规则。

## 淘宝 / 天猫（`taobao`，默认平台）

| 项 | 规范 |
|---|---|
| **主图比例/尺寸** | 1:1，生产建议 **800×800**（可放大至 1500×1500 供清晰缩放）；≥310×310 为上传下限。60+ 类目强制 800×800 |
| **主图数量** | 5 张套（部分类目 6 张）；**第 1 张为主投放图/搜索图** |
| **主图单张大小/格式** | ≤3MB，JPG/JPEG/PNG |
| **白底图** | 800×800，JPG，**38KB~300KB**，纯白底 **RGB(255,255,255)**，主体清晰完整、无过度 P 图；氛围背景元素颜色 ≤4 种。强制类目第 1 张或 5 张中至少 1 张为白底 |
| **主图视频** | 1:1 或 16:9，≤60s，≤30MB，MP4（部分类目强制） |
| **详情页** | 宽 **750px**（PC + 移动自适应），单张高建议 ≤1500px，JPG/PNG |
| **文字占比/牛皮癣** | 主图文字 ≤20%，**禁牛皮癣**：过度文字/水印/促销标签/边框/拼图/大 LOGO 遮挡主体。文字避让主体关键部位与右下价格标位 |

**天猫增量**（同 `taobao` 值，旗舰店更严）：白底强制类目更广、品牌 LOGO 位置/比例规范、牛皮癣打击更严、首图品质要求更高。

**调性**：功能参数 + 价格敏感；白底清晰、移动优先；信息层级服务「先看主体→卖点→钩子→点击/下单」。

**违禁**：极限词、假促销（虚构原价/倒计时/限量）、与实物不符、二维码/联系方式（部分类目）、「原单/外贸/专柜/同款/代购」类目词。

## 京东（`jd`）

| 项 | 规范 |
|---|---|
| **主图比例/尺寸** | 1:1，**800×800**（≥600×600 为上传下限），多数类目**白底强制** |
| **主图数量** | **6 张**（比淘宝多 1 张，含参数图/资质图位） |
| **主图单张大小/格式** | ≤1MB（白底图 <300KB），JPG |
| **白底图** | 800×800，<300KB，纯白 RGB(255,255,255)；强制类目广 |
| **主图视频** | 16:9，≤60s |
| **详情页** | 宽 **790px**，单张高 ≤1500px |
| **文字占比/牛皮癣** | 禁牛皮癣，禁促销标签遮挡主体 |

**调性**：参数详尽、资质齐全、服务标识（京东物流/自营）突出；白底规范、品质信任导向。

**违禁**：极限词、贬损竞品、**虚假比价**、**冒用「京东自营」**。

## 抖音电商（`douyin`）

| 项 | 规范 |
|---|---|
| **主图比例/尺寸（双套）** | **1:1**：≥600×600，白底 ≥800×800，推荐 800×800~1200×1200；**3:4**（服饰/鞋包/保健品等竖版类目）：≥375×500，推荐 600×800~750×1000。**首图必须实物图** |
| **主图数量** | 建议 ≥5 张；**主图不得完全相同内容**（需多角度/细节） |
| **水印** | 禁全屏水印、禁平台无关水印（美图秀秀/摄影网站等） |
| **商品/主图视频** | 9:16 1080×1920（短视频带货优先），或 1:1 |
| **详情页** | 宽 620~1290px（移动端 ~750px），高 <2000px（超出自动裁剪），≤20 张，单张 ≤1MB，JPG/PNG，推荐 16:9 或 21:9 长图。不同类目默认 1:1 或 3:4 |
| **文字/牛皮癣** | 禁画风低质、牛皮癣 |

**调性**：强情绪、强对比、利益前置；**信息流前 0.5 秒/首屏抓眼球**；竖屏优先；适合「场景 + 痛点 + 解决方案」。

**违禁**：极限词、**诱导好评/分享/关注/点赞返现**话术、虚假宣传。

## 小红书电商（`xhs`）

| 项 | 规范 |
|---|---|
| **主图/封面比例/尺寸** | **3:4 竖版优先**（原生 1080×1440），也支持 1:1 |
| **素材类型** | 商品卡图 vs 种草笔记图两种；商品视频 9:16 |
| **文字占比** | 弱硬广，文字克制，封面标题钩子强 |
| **数量** | 笔记型多图，商品卡按类目 |

**调性**：生活方式图文感、真实种草、**弱广告感/素人感**、情绪共鸣 + 软种草；封面标题位置（顶部/底部）按品类习惯。

**违禁**：极限词、**明显导流外部**、虚假素人感、诱导互动（与种草域重叠但电商转化向，须带合规软广边界）。

## 微信小店（`wechat_store`）—— 新增

> 微信小店 2024 年重启，规范仍在演进。以下为 2025 主流规则，上架前以[微信小店成长中心](https://store.weixin.qq.com/chengzhang/manual/shop/2)为准。

| 项 | 规范 |
|---|---|
| **主图比例/尺寸** | 1:1（最终商详页显示为**正方形**），推荐 **800×800**，≥480×480 |
| **主图数量** | 最少 **3 张**（食品饮料/生鲜类目最少 **4 张**），最多 **9 张**，**不得有重复图片** |
| **主图单张大小** | ≤10MB |
| **详情图** | 建议 **3-5 张**，多角度展示 |
| **商品视频** | 支持「商品主图引用视频号」（2025 新功能），9:16 竖屏 |
| **联动** | 与视频号/公众号联动是核心引流路径，素材可复用 |

**调性**：**信任背书 + 私域**、社交裂变导向、熟人推荐感；比传统平台更重「关系链信任」。

**违禁（微信生态极严）**：极限词、虚假宣传、**诱导分享/裂变话术**（「转发 X 次得 Y」「拉人头」等，比淘宝/抖音/京东更严）、欺诈营销。

## 通用（`general` / 未知 / 空，多平台兼容兜底）

- 主图 `1:1:2K`、详情 `3:4:2K`、封面 `16:9:2K`、分享 `1:1:2K` 或 `3:4:2K`、SKU `1:1:2K`。
- 文字占比 ≤20%、留白充足、信息层级清晰，**便于跨平台裁切复用**（同一套图裁出淘宝 1:1 / 抖音 3:4 / 小红书 3:4）。
- 严守《广告法》与各平台共性禁用词（详见 [prohibited-words-ecommerce.md](prohibited-words-ecommerce.md)）。

## 通用红线（所有平台）

- **文字占比**：主图 ≤20%，避免牛皮癣；文字避让主体与右下价格标位。
- **首屏可读**：移动端核心卖点落在首屏安全区（避开平台水印/价格标/导航栏）。
- **一致性**：产品跨图同一件；规格参数真实（与产品档案一致，是退货防线）。
- **资质真实**：认证/检测图标须有据，不得伪造；保健食品/特殊化妆品/医疗器械须有对应备案号。
- **白底诚实声明**：白底图的**纯白 RGB(255,255,255)** 与 **<300KB** 是**投放后期处理约束**，AI 生成无法保证像素级纯白与精确文件大小——生成时做到「白底为主、主体居中、无牛皮癣」，纯白与压体积由后期 `compress_image` / 裁切 / 合成完成。

## 类目维度

平台规范是平台级；**同一平台内不同类目（服饰 / 3C数码 / 食品 / 美妆个护 / 家居）的主图套系与详情页结构差异显著**（如服饰要平铺/挂拍/模特 + 3:4 竖图，3C 要参数表/认证图标，食品要保质期/生产日期展示位，美妆要成分表/特证）。详见 [category-specs.md](category-specs.md)。
