# 电商极限词与违禁词（电商版，独立于种草违禁词库）

> 适用《广告法》及主流电商平台（淘宝天猫/京东/抖音/小红书/微信小店）的**商业广告**文案合规。与种草笔记的「诱导互动」违禁词域不同，本库聚焦**极限化用语、虚假承诺、功效绝对化、贬损竞品、平台专属违规**。命中即高风险，须改写。

## 一、《广告法》极限词（绝对禁止）

不得在图内文字/文案中使用以下「最/极/首/独/级」类极限化用语：

- **「最」类**：最好、最大、最优、最低价、最便宜、最受欢迎、销量最高、全网最低、顶级、极品、极致、最强、最新（作广告宣称时）。
- **「一」类**：第一、第一品牌、唯一、首个、独家（无据）、第一名、销量第一、好评第一。
- **「级」类**：国家级、世界级、宇宙级、顶级、王牌、巅峰、王牌。
- **绝对/永久类**：100%、绝对、永久、永不、万能、包治、根治、药到病除、无效退款（无据承诺）、零风险。
- **夸张类**：神效、神奇、奇迹、史上、空前、绝版、稀有、珍贵（无据）。
- **排名/权威类**（无据）：中国名牌、驰名商标、国家指定、免检、特供、专供。

**改写方向**（用可证实的客观描述替代）：
- 「销量第一」→「月销 X 件」（有据）。
- 「全网最低」→「限时直降 X 元」「到手价 ¥X」。
- 「最好」→「针对 Y 场景设计」「X 项检测达标」。
- 「100% 纯棉」→「100% 棉（以水洗标/检测报告为准）」。
- 「永久」→「X 年质保」。

## 二、虚假承诺与功效绝对化（重点品类）

- **美妆/个护**：不得承诺「永久美白/祛斑/抗衰/根治痘痘/7 天换脸/淡化所有皱纹」等绝对功效；可用「有助于改善/提亮肤色（有据）」「质地滋润」。**美白/防晒/染发类属特殊化妆品，须有「国妆特字」备案号**。
- **健康/食品**：不得宣称「治疗/治愈/降三高/排毒/抗癌/壮阳」；**保健食品（蓝帽子）不得涉及疾病预防/治疗功能**，须标「本品不能代替药物」。普通食品不得宣称保健功能。
- **电子/数码**：不得虚标参数（假续航/假像素/假容量）；「防水」须标等级（IPXN）；「快充」须标功率。
- **通用**：不得「假促销」（虚构原价/虚假倒计时/虚假限量）、不得「无效退款」类无据承诺、不得伪造获奖/认证。

## 三、贬损竞品与不当比较

- 不得指名贬损具体竞品、不得伪造对比数据。
- 对比须有据（自有实验室/第三方报告），标注条件；可用「普通款 vs 本款」泛指，不用具体竞品名。

## 四、诱导违规（平台规则）

- 不得诱导好评/刷单/虚假交易（「好评返现」「好评送 X」「刷单」）。
- 不得诱导站外导流违规（按平台规则，如淘宝禁止微信号导流）。
- 不得伪造销量/评分/资质图标。

## 五、平台专属违规（按 `target_platform`）

| 平台 | 专属高风险词/行为 |
|---|---|
| **淘宝/天猫** | 「原单/外贸尾单/专柜同款/代购/专柜正品（无授权）」类目词；假促销；与实物不符 |
| **京东** | 冒用「京东自营/京东物流」（非自营不得标）；**虚假比价**（虚构划线价/历史最低）；贬损竞品 |
| **抖音** | 诱导「点赞/关注/转发/收藏返现」话术；「直播间承诺不兑现」；虚假宣传功效 |
| **小红书** | 明显导流外部（微信号/二维码）；**虚假素人感/软广未标**；诱导互动刷数据 |
| **微信小店** | **诱导分享/裂变**（「转发 X 次得 Y」「拉人头砍价」「帮砍一刀」类——微信生态比传统平台更严）；欺诈营销；外链诱导 |

## 六、特种类目资质与备案号（须对应，文案不得越界）

- **保健食品**：须有「国食健字 G/JXXXXXXXX」或蓝帽子标志；文案不得涉疾病预防/治疗，须标「不能代替药物」。
- **特殊化妆品**（美白/防晒/染发/烫发/祛斑/防脱/染发等）：须有「国妆特字 G202X XXXX」备案号。
- **医疗器械**：须有「械字号」注册/备案号；不得保证治愈。
- **食品/农产品**：须展示生产日期/保质期/生产厂家/执行标准；不得宣称疗效。
- **3C/电器**：须有 3C 认证/能效标识；参数须实标。

## 扫描与改写原则

- 命中**一/二/五/六类** → 高风险，必须改写并重生成相关图。
- 命中**三/四类** → 高风险，按规则处理。
- **上下文敏感**（避免误报）：
  - 「第一代/第一视角/第一印象」= 序数/客观陈述 → 放行；「销量第一/好评第一/第一品牌」= 极限化 → 命中。
  - 「极品/极地/臻致/金牌」为**注册商标或成分名** → 标注疑似误报、人工复核，不自动删核心信息。
  - 客观参数/认证编号陈述 ≠ 广告宣称 → 放行（但须有据）。
- 产品名/成分名本身含敏感字 → 疑似误报，标注人工复核，不自动删核心信息。
- 功效类品类（美妆/健康/电子）额外严查绝对化功效承诺 + 备案号格式。

| 原文（高风险） | 改写（合规） |

| 销量第一 | 月销 X+ 件（有据）/ 累计好评 X 万 |
| 永久美白 | 有助于提亮肤色（有据）/ 质地润泽 |
| 7 天换脸 | 持续使用 X 周，肤感更润（不承诺时效） |

| 国家级 / 免检 | 通过 XX 认证（具名且真实） |
