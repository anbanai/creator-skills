## 一、微信平台违禁类

### 1.1 政治敏感词

创作时完全回避以下类型内容，无合规替代方案：

- 现任/历任领导人姓名（用于负面语境或政治批评）
- 政治事件（六四、天安门事件等历史政治敏感节点）
- 分裂言论（台独、藏独、港独等分裂主张）
- 攻击中国共产党或国家政体的表述

**处理原则**：政治敏感内容不做替换，直接删除相关段落或更换选题。

### 1.2 色情低俗词

| 违禁词/表述 | 合规替代 |
|------------|---------|
| 裸露、赤裸、一丝不挂 | 删除或改为"自然状态" |
| 性感（用于人体描写） | 迷人的、有魅力的 |
| 挑逗、撩拨（性暗示语境） | 吸引、打动 |
| 约炮、一夜情 | 删除 |
| 床上技巧（性语境） | 删除 |

### 1.3 暴力恐怖词

| 违禁词/表述 | 合规替代 |
|------------|---------|
| 炸弹制作、爆炸物 | 删除 |
| 暗杀、行刺 | 删除 |
| 恐怖袭击（赞美/鼓励语境） | 删除 |
| 虐待、残忍折磨（详细描述） | 删除或改为"伤害" |
| 杀光、灭绝（针对群体） | 删除 |

### 1.4 赌博诈骗词

| 违禁词/表述 | 合规替代 |
|------------|---------|
| 博彩、赌场、赌博 | 删除 |
| 老虎机、百家乐、炸金花 | 删除 |
| 内幕消息、稳赚、躺赢 | 删除或改为"合理投资" |
| 刷单、薅羊毛（灰色手段语境） | 删除 |
| 洗钱、跑分 | 删除 |

### 1.5 虚假信息词

| 违禁词/表述 | 合规替代 |
|------------|---------|
| 据说可以治愈癌症 | 研究显示有助于辅助调理 |
| 经科学证明（无引用） | 研究表明（建议注明来源） |
| 100%有效、零副作用（未经认证） | 多数用户反馈效果良好 |
| 独家爆料（未经证实） | 有消息称（建议核实） |
| 绝密文件、内部泄露 | 删除 |

### 1.6 侵权相关词

| 违禁词/表述 | 合规替代 |
|------------|---------|
| [明星姓名]代言（无授权时） | 某知名艺人风格、明星同款 |
| [品牌名]官方推荐（无授权时） | 同类优质产品、[品类]推荐 |
| 未授权使用歌词/文学作品 | 改为自创内容或注明版权 |

## 二、广告法违禁类

### 2.1 绝对化用语（高频违禁）

以下词汇在广告/营销语境中严格禁止，需替换为相对表述：

| 违禁词 | 合规替代 |
|--------|---------|
| 最、最好、最强、最优 | 优质、出色、领先 |
| 第一、No.1（无权威认证） | 前列、深受好评 |
| 顶级、顶尖 | 高端、优质 |
| 极品 | 精品、甄选 |
| 国家级（无实际认证） | 专业级 |
| 世界级、全球最（无认证） | 国际品质、全球视野 |
| 首选（无数据支撑） | 热门选择、广受推荐 |
| 无与伦比、史无前例 | 卓越、出众 |
| 完美、完美无缺 | 精良、精心打造 |
| 终极、终极解决方案 | 系统化、全面 |

### 2.2 虚假承诺

| 违禁词 | 合规替代 |
|--------|---------|
| 保证效果、效果保证 | 追求良好效果 |
| 100%成功、百分百有效 | 高成功率、效果显著 |
| 永不失败、零失败 | 稳定可靠 |
| 零风险、无风险 | 风险可控 |
| 包治、包好、药到病除 | 有助于改善、辅助调理 |
| 立竿见影（医疗健康语境） | 效果明显 |
| 无条件退款（夸大承诺时） | 支持退款 |

### 2.3 医疗保健夸大

| 违禁词 | 合规替代 |
|--------|---------|
| 根治、彻底根治 | 有助于改善症状 |
| 特效、特效药 | 效果良好 |
| 祖传秘方、独家配方（夸大） | 传统配方、特色配方 |
| 治愈率100% | 改善率高 |
| 无副作用（未经认证） | 温和配方 |
| 国家认证（无实际认证） | 符合相关标准 |
| 医院推荐（无实际授权） | 专业建议参考 |

### 2.4 诱导消费词

| 违禁词 | 合规替代 |
|--------|---------|
| 秒杀（价格虚假对比） | 限时优惠 |
| 仅限今天（虚假紧迫感） | 近期优惠 |
| 错过不再有 | 数量有限 |
| 最后N件（夸大稀缺） | 库存有限 |
| 史上最低价（无依据） | 优惠价格 |
| 全网最低（无依据） | 极具竞争力的价格 |
| 免费（但有隐藏条件时） | 详见活动规则 |

### 2.5 权威性误导

| 违禁词 | 合规替代 |
|--------|---------|
| 国家推荐（无实际文件） | 业界认可 |
| 政府认证（无实际文件） | 符合相关规范 |
| 央视推荐（无播出记录） | 受到广泛关注 |
| 专家一致认为（无来源） | 业内专家认为 |
| 权威机构证明（无具体机构） | 参照行业标准 |
| 国际认证（无具体机构） | 参照国际标准 |

## 三、使用说明

### 检查优先级

1. **高风险**（直接删除/停止创作）：政治敏感、色情、暴力恐怖、赌博诈骗
2. **中风险**（必须替换）：广告法绝对化用语、虚假承诺、医疗夸大
3. **低风险**（建议替换）：诱导消费词、权威性误导

### 替换原则

- 优先使用表格中的合规替代词
- 无合规替代时，删除该词语或重写整句
- 替换后确保句意通顺、表达清晰
- 不得通过谐音字、拼音、特殊符号变相使用违禁词（如"最*好"）

### 报告格式

在内容检查报告中，按以下格式列出发现的违禁词：

```
违禁词检查报告：
- [词汇] → 已替换为 [合规表述]（位置：第X段）
- [词汇] → 已删除相关句子（位置：第X段）
共处理 N 处违禁词，内容已达到平台合规标准。
```
