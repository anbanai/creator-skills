## 封面的唯一使命

用户刷信息流时，决定是否点击的时间是 **0.5 秒**。封面不是内容摘要，不是视觉说明，它只做一件事：**让人在 0.5 秒内产生点击冲动**。

**三层分工**：
- 帖子标题：算法优化/搜索
- 封面：信息流点击率（CTR）
- 内容图：完读率

三者面向不同目标，不必传达相同信息。封面看到好奇就够了。

---

## 0.5 秒设计原则

- **极低信息密度**：只传一个信号，多了反而稀释注意力
- **强视觉对比**：主体和背景之间有清晰的明度/色彩对比
- **真实感优于精致感**：手机随拍感比设计感更能产生信任点击
- **利益前置**：把最大的好处放在最显眼的位置
- **文字权重最高**：封面主文案的字号要显著大于其他所有元素

## 7 种封面类型

| 类型 | 适用内容 | 设计要点 |
|------|----------|----------|
| **笔记感**（默认推荐） | 大多数内容 | 手写/圆润字体，实拍背景，手绘标注，低密度 |
| **高美感实拍** | 旅行、美食、穿搭 | 真实场景，人物入镜，构图留白，色调统一 |
| **拼贴感** | 好物合集、测评 | 多图拼贴，简洁网格，统一色框 |
| **海报风** | 活动、知识分享 | 全出血背景，几何构图，字体大胆 |
| **对比式** | Before/After，横向测评 | 分屏对比，对比强烈，结果直给 |
| **纯文字** | 干货、情感 | 文字即封面，背景简洁，字体个性 |
| **视频封面** | 视频笔记 | 关键帧截图，加文字叠层，竖版 |

### 2026 趋势补充（在 7 类基础上的强化变体）

这几种是 2026 信息流里点击率明显更高的封面形态，可视为上面 7 类的"增强版"，按内容择优：

| 趋势 | 适用内容 | 设计要点 |
|------|----------|----------|
| **大字报式** | 强利益点、干货结论 | 一句话利益 + 强对比色块，文字占比可到 40%（大字报是下方「文字占比 ≤30%」红线的有意例外，主打"一眼看到好处"） |
| **九宫格拼图** | 合集、盘点、好物榜 | 3×3 网格塞满产品/选项，统一色框，传递"量大管饱"的收藏感 |
| **氛围感实拍** | 情绪、生活方式、旅行 | 弱化文字，靠光影/色调/构图传递情绪，主打"想要这种生活"的代入 |
| **手账风批注** | 知识、攻略、避坑 | 实拍底图 + 手写体批注 + 箭头标注，真实感强、信息可信 |

### CTR 优先选型（不要一种打天下）

默认用"笔记感"，但按内容类型择优更能拉点击率：

- **合集 / 盘点 / 好物榜** → 九宫格拼图 或 拼贴感（视觉先传递"内容多、值得收藏"）
- **Before-After / 横向测评** → 对比式（结果直给，反差最强）
- **一句话强利益点（"7天搞定/不到百元/避雷清单"）** → 大字报式（利益前置，0.5 秒最易读出）
- **情绪 / 生活方式 / 旅行** → 氛围感实拍（靠代入感而非信息驱动点击）
- **教程 / 攻略 / 避坑** → 手账风批注 或 笔记感（真实可信，降低"广告感"）

选型原则：封面只做一件事——让人在 0.5 秒内产生点击冲动。信息留给标题（SEO）和内容图（完读）。

## 设计规范

**必须**：
- 字体：圆角/手写字体（不用衬线/黑体）
- 文字占比：≤30% 画面面积
- 关键词高亮：荧光标记或色块
- 焦点数量：单一焦点，≥50% 留白
- 背景：真实照片（非纯色/渐变/3D渲染）
- 标注：手绘风格批注
- 用户指定封面标题时，主文案必须直接使用该标题的核心短语
- 中文主题的封面只允许出现简体中文，禁止英文、拼音、翻译和伪词

**禁止**：
- 3D 渲染/合成感背景
- 纯色大背景
- 粗黑加粗字体
- 对称 PPT 式布局
- 密集文字墙
- 手机框/设备框嵌套
- 过度磨皮精修
- 与标题无关的季节、品类或产品
- 模糊养生承诺（如治疗、治愈、立刻变好）

## Prompt 模板（v2 - 中文文字渲染优化）

```
{visual_subject_description}

风格要求：{style}

平台规范：
- 3:4 竖版构图，种草笔记信息流封面
- 笔记手账风格，真实感而非设计感
- 单一主视觉焦点，50% 以上留白营造呼吸感
- 实拍质感背景（勿用纯色/3D/渐变）

【文字要求 — 重要，违反则视为生成失败】
画面中央偏上必须显示中文主文案，使用圆角手写体，字号占图宽 12-15%，
深色字配浅色背景或白色字配深色背景（保证对比度）。
主文案内容（必须原样渲染，禁止修改任何字符、增删字、改标点）：
「{required_text}」

【禁止项 — 严格遵守】
- 任何英文字母、拼音、阿拉伯数字编号
- 乱码、伪词、错别字、字形错误
- 在主文案区域画装饰物遮挡文字
- 与主文案无关的额外中文文字
- 模糊养生承诺（治疗、治愈、立刻变好等）

内容主题：{topic}
视觉主体（不能省略，必须能直接看出主题）：{visual_subject}
```

**生成方式**：

通过 MCP 工具调用 `generate_image`，image_type 设为 `"cover"`。
