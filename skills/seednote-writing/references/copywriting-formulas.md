## 1. 美妆护肤时尚类

**文案公式**：个人情况 + 解决痛点 + 产品亮点 + 个人感受 + 使用效果

**写作要素**：
- 先描述多数人会遇到的化妆问题或皮肤状态（引起共鸣）
- 推出产品，突出产品优点与特点
- 搭配图片描述使用效果
- 最后详细描述使用方法
- 语气幽默风趣，添加对话感，增加感官感受词汇（增强代入感）

**内容真实性**：保证内容真实性和可操作性，确保所描述功能的真实性

**人设参考**：专业彩妆达人——出新品化妆教程、明星仿妆、口红试色等

## 2. 家居家装、日用百货类

### 家装类

**文案公式**：装修背景 + 装修亮点 + 热门区域 + 服务点 + 装修效果 + 个人感受

**写作要素**：
- 抛出装修风格、户型或装修过程中的小设计
- 给出解决方法，展示专业性
- 引导私信、评论区讨论（增加转化）
- 详细给出价格及操作方法，注意区分装修风格

**人设参考**：专业设计师人设，塑造品牌口碑和方案展示

### 日用百货类

**文案公式**：产品亮点 + 个人使用感受 + 案例种草

**写作要素**：
- 描述日常生活中经常遇到但没有合适解决方法的问题
- 推出产品，描述如何精准解决该问题
- 呼吁大家囤货购买
- 给出价格、购买获得方法、功能实用性详细介绍

**人设参考**：不需要特别人设（产品线多时），以真实、真诚、真心的分享圈粉

## 3. 旅游餐饮类

**文案公式**：目的地 + 卖点 + 价格 + 衣食住行 + 注意点

**写作要素**：
- 详细描述可玩性、性价比、餐饮住宿
- 使用真情实感的体验式语句（如"和爱的他坐在洱海边吹风"）
- 行文逻辑：按时间线索展开，或分地点区域介绍
- 幽默风趣感与逻辑清晰并重

**合集类笔记**：突出每个地点的特点，保持主题统一（如"爱吃辣的集美不能错过的几个城市"）

**人设参考**：专业旅游达人——提供地道玩法和鲜为人知景点；按节日/季节推荐热门目的地

## 4. 3C 及电器类

**文案公式**：个人需求点 + 产品卖点 + 问题解决情况 + 个人感受

### 方向一：专业测评

- 站在专业测评角度，使用专业术语
- 对电器各方面进行专业点评（性能、功能详细介绍）
- 关注用户需求（如洗衣机：波轮滚轮选择、高温除菌、是否洗烘一体）

### 方向二：日常好物分享

- 以日常好物分享类型出现，保证体验真实性
- 推荐的电器与账号身份定位一致
- 描述日常使用遇到的问题 → 推出产品 → 描述如何解决 → 特有优势 → 呼吁购买

**人设参考**：轻奢美容仪类——目标 30+ 一二线有消费能力女性，语气生活化，容易吸引目标人群

## 5. 摄影写真类

**文案公式**：背景+选择原因 → 详细流程（价格/地点/服务/摄影师等） → 真实感受+推荐理由

**写作要素**：
- 前 2 行介绍自己背景和选择原因（种草已久/朋友安利/明星推荐等）
- 主内容按服务流程分享，细节到价格、地点、前台服务、化妆师、摄影师、小助理
- 最后总结真实感受、推荐原因和存在的缺陷
- 明确产品风格（浪漫法式/中式古典等）；明确可提供服务（如 8 服 8 造）
- 排版清晰，可搭配 emoji 突出重点

**真实性**：样片和服务必须具有专业度，才能让用户产生信赖并转化

**人设参考**：
- 高端定制：在个性签名和每篇笔记中都提及卖点
- 简约中小型：根据拳头产品写笔记，更新频率稳定，标题口语化贴近生活

## 6. 教育培训、金融行业

**文案公式**：个人背景介绍 + 引起共鸣的痛点 + 产品亮点 + 现状和感受

### 方向一：个人经验分享

- 具备强烈个人内容特点 + 普适性 + 与账号身份定位结合
- 描述自己情况（如"二本逆袭考研上 985"）→ 日常生活记录（如笔记方法、文具好物）→ 推广服务 → 引导转化
- 让有相同习惯的用户产生共鸣，同时吸引向往该群体的用户

### 方向二：专业知识分享

- 确保内容正确性和专业性，言之有物
- 逻辑性强，让用户对专业度产生信赖
- 描述受众常遇到的问题 → 单刀直入知识科普 → 展示专业能力 → 实现转化

**人设参考**：专业靠谱的理财老师——内容垂直、账号定位清晰，吸引关注资产增值的用户

---

## 通用注意事项

1. **风格一致性**：文案与封面、标题风格一致，与账号整体风格一致
2. **对话感与代入感**：不必刻意变得有趣，适当加入调侃，站在用户角度思考
3. **真实可信**：所有描述的功能和效果必须真实，避免虚假承诺
4. **人设持续输出**：商家在每篇笔记中都要体现人设定位，增强账号垂直度
