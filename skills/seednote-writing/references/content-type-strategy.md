## 结构模板详解

### 编号清单式（知识干货/好物推荐）

```
[钩子：痛点场景 1-2 句]

1️⃣ [要点一] + 简短说明（1-2 句）
2️⃣ [要点二] + 简短说明（1-2 句）
3️⃣ [要点三] + 简短说明（1-2 句）

💡 [补充提示或注意事项]

[互动收尾：评论引导/收藏暗示]
```

### 叙事沉浸式（情感生活/旅行记录）

```
[场景开头：感官细节 1-2 句]

[情感发展：具体事件或感受 2-3 句]

[感悟或转折：核心观点 1-2 句]

[氛围收尾：留白或余韵]
```

### 表格对比式（测评对比）

```
[钩子：测评结论预告 1 句]

[产品A] ✅ 优点 / ❌ 缺点
[产品B] ✅ 优点 / ❌ 缺点

[推荐结论：什么人选什么]

[互动收尾：选择题引导评论]
```

### 步骤流程式（教程攻略）

```
[钩子：效果承诺 1 句]

📍 Step 1: [具体操作]
📍 Step 2: [具体操作]
📍 Step 3: [具体操作]

💡 [注意事项或进阶提示]

[互动收尾：收藏暗示]
```

## Emoji 风格规则

每种内容类型对应一个 emoji 集，从该集合中选取使用。不混用不同集合的 emoji。

**使用密度**：
- 干货类：每段 1-2 个，结构化使用（段落开头 + 关键词旁）
- 情感类：关键处点缀（2-4 个/全文），不过度
- 测评类：对比标记用，保持清晰（✅❌ 成对使用）
- 美食类：食物 emoji 增加感官体验

**使用位置**：
- 段落开头：作为视觉锚点引导注意力
- 关键词旁：强调核心信息
- 不在句末连续堆叠（避免 emoji 墙）
